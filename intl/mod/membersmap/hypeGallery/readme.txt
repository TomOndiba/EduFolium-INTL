PLUGIN DESCRIPTION
------------------
hypeGallery is an attempt to create an alternative to TidyPics that offers more intuitive user interface and ajax funcationalities.

Main features include:
-- Collaboration albums - users can allow other users to add photos to their galleries
-- Geolocation of albums and images (hypeMaps required)
-- Inline photo tagging
-- Capability to set images as profile icons
-- Image downloads
-- One click set-album-cover

REQUIREMENTS
------------
1) Elgg 1.8.3+
2) hypeFramework 1.8.5+

INTEGRATION / COMPATIBILITY
---------------------------
-- hypeGallery integrates with hypeMaps, hypeComments
